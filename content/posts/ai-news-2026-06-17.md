# 🔥 오늘의 핵심 트렌드

오늘의 흐름은 **"자기진화 에이전트의 신뢰성"** 으로 수렴한다. 멀티모달 평가 루프에서 평가자 선호가 단일 전략에 집중되는 EPC 현상이 수치로 입증됐고, 에이전트가 숨겨진 환경 모델을 얼마나 효율적으로 추론하는지를 DFA 기반으로 정밀 측정한 연구도 등장했다. RL 알고리즘 측에서는 REINFORCE부터 GRPO까지 전체 계보를 단일 목적 함수 2개 축으로 통합한 서베이 프레임워크가 나오면서, 실무자들이 알고리즘 선택 기준을 처음으로 공리적으로 따질 수 있게 됐다. 도구 측에서는 Claude Code v2.1.179·동적 워크플로우·중첩 서브에이전트가 오늘 주목할 세 가지다.

## [1] Multimodal Evaluator Preference Collapse: Cross-Modal Contagion in Self-Evolving Agents

- **저자/소속** Zewen Liu (Qilu Institute of Technology, China), **arXiv** 2606.16682, **카테고리** cs.AI / cs.CL
- **핵심 기여** 자기진화 에이전트의 멀티모달 평가 루프에서 단일 전략이 가중치를 독식하는 EPC(평가자 선호 붕괴) 현상이 텍스트 전용보다 3.2× 강하게 발생함을 입증. 텍스트 선호가 시각 태스크 선택까지 오염시키는 교차 모달 오염(cross-modal contagion)을 최초 계량화.
- **방법론 요약** GPT-4o를 평가자, DeepSeek-chat을 에이전트로 삼아 텍스트·시각 혼합 피드백 루프 구성. 4단계 격리 훈련으로 모달리티별 선호 형성 과정 추적. 오염 계수 수치화 + 전략 역전 문서화. 53회 독립 반복, 15,592 API 호출로 통계 검증.
- **주요 결과** 단일 전략(`step_by_step`) 가중치 **48.4%** 독점(텍스트 전용 대비 **3.2×**); 시각 3개 전략 합계 9.1%에 불과

## [2] Can LLM Agents Infer World Models? Evidence from Agentic Automata Learning

- **방법론 요약** 에이전트가 숨겨진 DFA를 멤버십 쿼리("이 문자열이 목표 언어에 속하는가?")와 동치 쿼리("이것이 목표 DFA인가?") 두 종류만으로 추론. DFA 상태 수로 복잡도 조절. 추론 모델 vs 비추론 모델 대조. 실패 궤적 정성 분석.
- **주요 결과** DFA 크기 증가 → 성능 급격 하락; 추론 모델 현저히 우수; 주요 실패 유형: 쿼리 계획 미흡 / 증거 통합 부족 / 가설 구성 오류
- **🛠 실무 시사점**

  | 에이전트 아키텍처 | 쿼리 계획 컴포넌트 명시적 설계 권장 |

## [4] A First-Principles Derivation of LLM Policy Optimization: From Expected Reward to GRPO and Its Structural Extensions

- **저자/소속** 저자 정보 논문 직접 확인 권장, **arXiv** 2606.16733, **카테고리** cs.LG / cs.AI
- **핵심 기여** REINFORCE → PPO → GRPO → 에이전틱 RL 파생 알고리즘 전체를 목적 함수 J(θ) = E[R(τ)]의 2개 인자(궤적 확률 × 보상)를 각 알고리즘이 어떻게 수정하는지로 통합 설명. 알고리즘 설계 선택의 수학적 필연성을 추적할 수 있는 진단·확장 가능 프레임워크.
- **방법론 요약** 두 축 분류: (1) 궤적 확률 수정 방향, (2) 보상 형태 수정 방향. 각 알고리즘이 앞선 방법의 어떤 실패를 해결하는지 계보화. GRPO = 학습된 비평자 없이 그룹 상대 보상 정규화로 대체한 구조를 수식 도출. 에이전틱 RL 확장까지 포함.
- **주요 결과** 통합 프레임워크 제공 — REINFORCE→PPO→GRPO 계보의 수학적 필연성 명시; 새 알고리즘 설계 시 2축 좌표계로 체계적 탐색 가능
- **🛠 실무 시사점**

  | 역할 | 활용 |
  |---|---|
  | ML 엔지니어 | GRPO 하이퍼파라미터 튜닝 이유를 수식으로 이해 |
  | 연구자 | 새 RL 알고리즘 설계 시 놓친 차원 탐색 |
  | PM / 팀장 | 팀의 RL 알고리즘 선택 근거 설명 자료 |

  도입 난이도 🟢 · 도입 시기: RLHF/GRPO 도입 검토 전 배경 학습

- **한계** 실험 결과 없는 이론·서베이 논문; 새 알고리즘 미제안; 발표 시점 이후 연구 미포함
- **링크** [arxiv.org/abs/2606.16733](https://arxiv.org/abs/2606.16733)

- **저자/소속** Abbas Mammadov, Iskander Azangulov, Yee Whye Teh (University of Oxford); Özgür Kara, James Matthew Rehg (UIUC); Hyungjin Chung (EverEx); 외, **arXiv** 2606.17048, **카테고리** stat.ML / cs.CV
- **핵심 기여** 확산·플로우 모델로 선형 역문제를 풀 때 필요한 사후 점수를 **가우시안 보간 하에서 닫힌 형태로 정확히 도출**. 기존 근사 추정의 오차를 수학적으로 제거하며, 사후 샘플링이 비등방 노이즈 공분산 하에서 표준 디노이징 문제로 환원됨을 증명.
- **방법론 요약** EPS(Exact Posterior Score) 훈련 목적 함수 도출. 기존 사전 훈련의 입출력 구조 그대로 보존. 선형 가우시안 역문제의 사후 점수 = 비등방 노이즈 하의 이동 피벗 디노이징 문제임을 수식 증명. 기존 확산·플로우 파이프라인 재활용 가능.
- **주요 결과** 기존 근사→EPS 정확 도출; 표준 디노이징 파이프라인 재사용으로 추가 구조 변경 없이 선형 역문제 전반 적용 가능
- **🛠 실무 시사점**

  | 적용 분야 | 기대 효과 |
  |---|---|
  | 의료 영상 (MRI, CT 복원) | 정확한 사후 분포로 불확실도 정량화 |
  | 지진파 역산 | 지하 구조 추론 오차 감소 |
  | 이미지 복원 (SR·인페인팅) | 기존 훈련 파이프라인 재사용으로 도입 비용 최소화 |

  도입 난이도 🔴 · 도입 시기: 확산 모델 기반 역문제 연구·제품팀

- **💡 핵심 아이디어** `/cd` 하나로 현재 세션을 다른 작업 디렉토리로 이동. 세션 재시작 없이 프롬프트 캐시를 그대로 유지하며 이동하고, 새 디렉토리의 `CLAUDE.md`는 시스템 프롬프트 교체 없이 메시지로 추가돼 맥락이 끊기지 않는다.

  ```text
  /cd ../other-project
  ```
